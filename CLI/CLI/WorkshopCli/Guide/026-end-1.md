Agora vamos preparar o Boss.
O Boss e o inimigo grande do final do jogo.

Antes de ele aparecer, precisamos de carregar as imagens e guardar algumas informacoes.

1. Procura a funcao:
   [/color] [color=blue]
    function love.load()
   [/color] [color=white]
2. Vai ate ao fim dessa funcao, antes do ultimo "end".

3. Cola este codigo nesse sitio:
   [/color] [color=blue]
    require("boss")

    bossImagem = love.graphics.newImage("inimigo.png")
    ImagemTirosBoss = love.graphics.newImage("inimigo1.png")
    boss = nil
    bossVida = 10
    bossApareceApos = 5
    inimigosDerrotados = 0
    bossDisparoTempo = 0
    bossDisparoTempoMax = missilTempoMax + 1
   [/color] [color=white]
O que isto faz:

- require("boss") chama o ficheiro boss.lua.
- bossImagem guarda a imagem do Boss.
- boss = nil quer dizer que o Boss ainda nao apareceu.
- bossVida = 10 da 10 vidas ao Boss.
- bossApareceApos = 5 quer dizer que ele aparece depois de derrotares 5 inimigos.
- inimigosDerrotados = 0 comeca a contagem em zero.

4. Agora vai ate ao fim do ficheiro "main.lua".
Depois do ultimo "end", cola esta funcao nova:
   [/color] [color=blue]
    function criaBoss()
        boss = Boss:new(
            bossImagem,
            love.graphics.getWidth() - 200,
            love.graphics.getHeight() / 2 - 100,
            bossImagem:getWidth(),
            bossImagem:getHeight(),
            50,
            50,
            bossVida
        )
    end
   [/color] [color=white]
Esta funcao ainda nao mostra o Boss.
Ela so ensina o jogo a criar o Boss quando precisarmos dele.

Atencao:

- O primeiro bloco fica dentro de "love.load()".
- A funcao "criaBoss()" fica fora das outras funcoes, no fim do ficheiro.
- Nao apagues o codigo que ja tinhas.

No proximo passo vamos fazer o Boss aparecer no ecra.
