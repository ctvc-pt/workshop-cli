Agora o Boss aparece, mas ainda e invencivel.
Vamos fazer com que ele perca vida quando um missil lhe acerta.

1. Procura a funcao:
   [/color] [color=blue]
    function verificaMissilInimigoColisao()
   [/color] [color=white]
2. Vai ate ao fim dessa funcao, mas fica antes do ultimo "end".

3. Cola este codigo nesse sitio:
   [/color] [color=blue]
    if boss then
        for index, missil in ipairs(misseis) do
            if intercepta(missil.posicaoX, missil.posicaoY, missil.width, missil.height, boss.x, boss.y, boss.width, boss.height) then
                boss:levaDano(1)
                table.remove(misseis, index)
                if boss:perde() then
                    boss = nil
                    pontuacao = pontuacao + 1000
                    estadoJogo = "vitoria"
                end
                break
            end
        end
    end
   [/color] [color=white]
O que este codigo faz:

- Primeiro ve se o Boss existe.
- Depois olha para todos os misseis.
- Se um missil tocar no Boss, o Boss perde 1 vida.
- Esse missil desaparece.
- Se a vida do Boss chegar a zero, ganhas o jogo.

Atencao:

- Este codigo fica dentro da funcao "verificaMissilInimigoColisao()".
- Cola antes do ultimo "end" dessa funcao.
- Nao apagues o codigo que ja verifica misseis contra inimigos normais.

Guarda e testa com Alt + L.
O Boss tem 10 vidas. Quando acertares 10 vezes, deve aparecer a vitoria.
