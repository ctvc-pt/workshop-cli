Aqui vao algumas ideias para melhorares o teu jogo:

1. Muda a "velocidade" para tornar o jogo mais rapido
   ou mais lento

2. Muda "jogador.forcaSalto" para o skater saltar mais
   alto ou mais baixo

3. Muda o "intervalo" para os obstaculos aparecerem mais
   ou menos vezes

4. Tenta fazer a velocidade aumentar com o tempo!
   Dica: na funcao update, adiciona algo como:
[/color] [color=blue]
    velocidade = velocidade + dt * 2
[/color] [color=white]

5. Adiciona um recorde pessoal (melhor pontuacao)!
   Dica: cria uma variavel "melhorPontuacao" no love.load()
   e atualiza-a quando o jogo acaba.

Diverte-te a experimentar e mostra o teu jogo aos colegas!
