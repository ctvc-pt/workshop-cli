# Things to do manually

`Install.bat` handles .NET 8, Python, Git, Visual Studio Code and the required VS Code extensions automatically. Only the steps below still need to be done by hand:

1. Install `Ollama`, but do not install any models.
2. Right click on cli shortcut on the desktop, click on `Properties`, go to `Compatibility`, and select the `Run this program as an administrator` option.

## No Issues

If the person closes the CLI prompt, just run the CLI shortcut that is created on the desktop.
